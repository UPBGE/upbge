UPBGE JavaScript/TypeScript API

Welcome to the UPBGE JavaScript/TypeScript API documentation.

UPBGE supports JavaScript and TypeScript as alternative scripting languages to Python, using the V8 JavaScript engine. This allows developers to write game logic in JavaScript/TypeScript with the same functionality as the Python API.
